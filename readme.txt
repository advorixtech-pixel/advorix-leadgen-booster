== Description ==

Advorix LeadGen Booster is a powerful and lightweight WordPress popup lead capture plugin designed to increase conversions and grow your email list effortlessly.

This free WordPress lead generation plugin allows you to display a fully responsive popup form with Name and Email fields along with a floating WhatsApp button for instant customer engagement.

Perfect for bloggers, business websites, marketing agencies, landing pages, and eCommerce stores.

Key Features:

* Responsive Popup Lead Form
* WhatsApp Floating Button
* Admin Dashboard for Lead Management
* Secure Data Handling
* Mobile Optimized Design
* Lightweight & Fast Loading
* Beginner Friendly Setup
* Works with All WordPress Themes

Why Choose Advorix LeadGen Booster?

Unlike heavy popup plugins, this plugin is optimized for speed, performance, and higher conversion rates. It helps you capture leads without slowing down your website.

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure popup settings from the admin dashboard.

== Frequently Asked Questions ==

= Is this plugin free? =
Yes, this is a completely free lead generation plugin.

= Does it work on mobile devices? =
Yes, the popup and chat button are fully responsive.

= Where are leads stored? =
Leads are stored securely in the WordPress database.

= Does this plugin collect personal data? =
Yes. It stores lead names and email addresses submitted by visitors. The plugin integrates with WordPress personal data export and erase tools.

== Privacy ==

This plugin stores submitted lead data (name, email, date) in a custom database table.
Site owners are responsible for obtaining consent and publishing an appropriate privacy policy.
